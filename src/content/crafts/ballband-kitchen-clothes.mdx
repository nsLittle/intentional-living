instructions: |
  Cast on: With A, CO 49 sts.
  Row 1: Using A, knit.
  Row 2: Purl.
  Row 3: Knit.
  Row 4: Purl.
  Row 5: Join B, k6, slip 1 purlwise *k5, sl1 purlwise; rep from * to last 6 sts, k6.
  Row 6: K2, yarn forward (yf), slip 1 purlwise, yarn back (yb), *k5, yf, slip 1 purlwise, yb; rep from * to last 2 sts, k2.
  Row 7: P6, yb, slip 1 purlwise, yf, *p5, yb, slip 1 purlwise, yf; rep from * to last 6 sts, p6.
  Row 8: Repeat Row 6.
  Row 9: Using A, knit.
  Row 10: Purl.
  Row 11: Using B, k3, slip 1 purlwise, *k5, slip 1 purlwise; rep from * to last 3sts, k3.
  Row 12: K3, yf, slip 1 purlwise, yb, *k5, yf, slip 1 purlwise, yb; rep from * to last 3sts, k3.
  Row 13: P3, yb, slip 1 purlwise, yf, *p5, yb, slip 1 purlwise, yf; rep from * to last st, p3.
  Row 14: Repeat Row 14.
  Row 15: Knit.
  Row 16: Purl.
  Repeat: Repeat rows 5-16 eleven more times.
  Repeat: Repeat rows 5-10 once.
  Repeat: Repeat rows 9-10 once more.
  Bind off.
